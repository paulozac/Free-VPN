# ZacVPN Support

## Need Help?

If you're experiencing issues with ZacVPN or have questions, we're here to help.

### Common Questions

**Q: How do I add a VPN profile?**
Press "Upload Profile" on the main screen, then scan the QR code with your phone. You'll be taken to a web page where you can upload your WireGuard (.conf) or OpenVPN (.ovpn) configuration file.

**Q: Where do I get a VPN configuration file?**
ZacVPN is a client only — it does not provide VPN servers. You need to obtain a configuration file from your own VPN server or a VPN provider that supports WireGuard or OpenVPN.

**Q: The VPN connects but I have no internet.**
Make sure your VPN server is correctly configured and running. Check that your configuration file includes valid DNS settings.

**Q: Can I use multiple VPN profiles?**
Yes. Upload as many profiles as you need and switch between them from the profile list. If you switch while connected, ZacVPN will automatically reconnect to the new server.

**Q: Does ZacVPN collect my data?**
No. ZacVPN collects zero data. No analytics, no accounts, no tracking. See our [Privacy Policy](PRIVACY_POLICY.md) for details.
